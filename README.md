# Cheon_hw4_nestedgrid, flexbox
This repo contains the submission for MMED-1056 Homework 4 - Nested Grid and Flexbox by Jin Cheon. 
The project creates a responsive hero section using CSS Grid, Nested Grid, and Flexbox for a clean and organized layout. 
Grid is used for the main structure, nested grid for text, and Flexbox for aligning elements like buttons.

The design adapts to different screen sizes with responsive images, and the CSS order property is used to adjust layout without changing the HTML.

## Installation 💻
No installation required, simply download the zip file from Github link and extract.

## Usage
Open index.html in your choice of browser to view and enjoy my project.
